# LabAC_PI1
O projeto busca o desenvolvimento de um software voltado para a gestão interna de processos laboratoriais em um Laboratório de Análises Clínicas.

## Tecnologias

- Site: **React** para frontend rápido e responsivo e **Flask** para o backend.
- Banco de Dados: **Supabase** para armazenar dados dos usuários.
- Controle de versão: **Git + GitHub** para colaboração e histórico.
- Design e Diagramas: **Miro**, **Lucid** e **Figma**.
- Deploy: **Vercel (Front-End)** e **Render (Back-End)**.
